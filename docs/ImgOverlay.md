# RectOverlay图层

### 概述

增加mouseover更换图标

[实践案例](https://smartdata.b0.upaiyun.com/inmap/examples/ImgOverlay-formatter.html ':include :type=iframe width=100% height=600px')


### styleOption

| 状态        | 类型   |  说明  |
| --------   | -----:  | :----:  |
| mouseOver    | Object |   鼠标悬浮状态的样式配置    |


### 样式描述

| 参数        | 说明   |  类型  |
| --------   | -----:  | :----:  |
| icon    | 图片绝对路径地址后者图片对象 |   path/Image    |
