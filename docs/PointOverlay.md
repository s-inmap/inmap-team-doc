# PointOverlay图层


### eventOption

| 参数        | 说明   |  类型  |  默认值  |
| --------   | -----:  | :----:  | :----:  |
| onMouseEnter    | 鼠标进入被选元素时。 
item:当前鼠标悬浮选中的数据对象。 
event: 事件对象 
this: 当前对象 |   Function    |     -     |
| onMouseLeave    | 鼠标离开被选元素时。 
item:当前鼠标悬浮选中的数据对象。 
event: 事件对象 
this: 当前对象 |   Function    |     -     |
